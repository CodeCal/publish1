Query Selector

document.querySelector()


you only get back the first item that matches your query


  • use '#' to retrieve via ID
  • use '.' to retrieve via class
  • use space between two selectors for hierarchical selection (e.g. ...querySelector("li a"). There's a space in between to denote a hierarchical relationship
  • use no space between two selectors to be more specific (e.g. ...querySelector("li.a"). No space in between to denote the specific list item that also has a hyperlink/anchor tag
